# Gushen AI 帖子风格设计指南 (Post Style Guide)

## 核心设计哲学：化数据为故事

我们的最终用户是广大散户，他们需要的是**能看懂、有启发、愿意读**的内容，而不是一份冷冰冰、格式化的研究报告。

**核心理念**：将专业的数据分析，用"讲故事"的方式包装起来，赋予冰冷的数据以"人格"和"剧情"，让用户像看一场精彩的"资金对决大戏"，而不是在读一篇枯燥的"研究报告"。

---

## 帖子结构模板

### 1. 标题：戏剧化、抓眼球

**原则**：用生动、有冲突感、甚至略带夸张的词汇，立刻构建出一个戏剧化场景，最大程度激发用户的好奇心。

**格式**：
```
《[个股名称] + [核心事件]！"[主角A]" + [行为A]， "[主角B]" + [行为B]！》
```

**示例**：
> **原始标题**：厦门信达 (000701.SZ) 20250617 龙虎榜分析报告
> **优化后标题**：**《厦门信达高位跳水！"T王"拉萨天团独木难支，神秘机构2800万胜利大逃亡！》**

---

### 2. 文章开头：AI画重点（结论先行）

**原则**：尊重散户时间宝贵、缺乏耐心的特点，在文章开头用最凝练的语言给出当天战局最核心的结论。

**格式**：
> **【AI一句话看盘】**
> [个股名称]今日上演[战况描述]，**[核心原因/关键角色]是导致[股价结果]的核心原因**。[其他关键角色]虽[对应行为]，但终究[结果]。短期趋势[看多/看空]，风险[加剧/降低]。

**示例**：
> **【AI一句话看盘】**
> 厦门信达今日上演多空大战，**机构主力坚决出货是导致股价大跌的核心原因**。"T王"虽率领东方财富拉萨天团奋力做T自救，但终究势单力薄，未能扭转颓势。短期趋势转空，风险加剧。

---

### 3. 核心内容：故事化叙事

#### 3.1 战场风云：今天发生了什么？

**原则**：解释上榜原因，将专业的龙虎榜术语"翻译"成散户能理解的大白话，并点明其市场含义。

**格式**：
> **【今日战况：为何登上龙虎榜？】**
> [个股名称]今日以[具体涨跌幅]接近[涨停/跌停]，直接触发了"[上榜原因]"的顶级警报。这通常意味着[原因的通俗化解释]，是市场最强烈的[看多/看空]信号之一。

**示例**：
> **【今日战况：为何登上龙虎榜？】**
> 厦门信达今日以-10.04%接近跌停，直接触发了"日跌幅偏离值达到7%"的顶级警报。这通常意味着市场上出现了巨大的抛压，有大资金在不计成本地卖出，是市场最强烈的看空信号之一。

#### 3.2 多空对决：谁在买？谁在卖？

**原则**：将买卖双方席位"人格化"，赋予其生动的"角色"或"标签"，并用对比手法强化冲突感。

**格式**：
> **【主角登场：两大阵营针锋相对】**
> 
> **⚔️ 空方主力（卖方）：[角色描述的卖方标签]**
> *   **主角**：[卖方核心席位/群体名称]
> *   **动作**：净卖出高达 **[金额]**！
> *   **AI解读**：这是今天[行为，如：砸盘]的绝对主力！在[股价位置，如：历史高位]附近，如此巨大的卖出，[行为解读，点明意图]。
>
> **🛡️ 多方主力（买方）：[角色描述的买方标签]**
> *   **主角**：[买方核心席位/群体名称]
> *   **动作**：合计净买入约 **[金额]**。
> *   **AI解读**："[主角名称]"今天显然在执行他最擅长的"[战术，如：做T]"战术。[行为解读，点明意图和实力对比]。

**示例**：
> **【主角登场：两大阵营针锋相对】**
> 
> **⚔️ 空方主力（卖方）：胜利大逃亡的"神秘机构"**
> *   **主角**：机构专用席位
> *   **动作**：净卖出高达 **2,868万元**！
> *   **AI解读**：这是今天砸盘的绝对主力！在股价历史高位附近，如此巨大的卖出，无论是调仓还是获利了结，都表明其不再看好短期走势，出逃意愿十分坚决。
>
> **🛡️ 多方主力（买方）：奋力自救的"T王天团"**
> *   **主角**："T王"率领的东方财富拉萨天团（旗下四大营业部联手）
> *   **动作**：合计净买入约 **1,042万元**。
> *   **AI解读**："T王"今天显然在执行他最擅长的"做T"战术，通过高抛低吸来降低成本。虽然买入金额不小，但和机构的抛压相比，如同杯水车薪。这更像是一种自救行为，而不是主动进攻。

#### 3.3 战術復盤：AI深度洞察

**原则**：将结构化的分析点（如集中度、协同性、历史背景）提炼为几个关键的"洞察"，用通俗易懂的语言揭示战局背后的逻辑。

**格式**：
> **【AI战术复盘：揭秘背后的玄机】**
> 1.  **"[洞察点1的标题]"**：[对洞察点1的具体解释]。
> 2.  **"[洞察点2的标题]"**：[对洞察点2的具体解释]。
> 3.  **"[洞察点3的标题]"**：[对洞察点3的具体解释，特别是结合历史走势]。

**示例**：
> **【AI战术复盘：揭秘背后的玄机】**
> 1.  **"T王"是团伙作战，而非单打独斗**：买方前四名清一色是东方财富拉萨的席位，这是典型的协同作战信号，证明"T王"并非一个人，而是一个风格统一的游资团体。
> 2.  **力量对比悬殊，机构掌握主动权**：机构卖出的2868万，远大于游资买入的1042万。这说明市场的主动权牢牢掌握在卖方手中，游资的承接力量完全不足以消化抛压。
> 3.  **历史位置是关键**：别忘了，在这次大跌前，股价已经从5.33元暴涨到8.35元（涨幅**+56.6%**）。在如此高的获利盘面前，任何风吹草动都可能引发踩踏。今天的机构出货，就是点燃火药桶的那根火柴。

---

### 4. 结尾：策略与风险（提供价值）

**原则**：给出明确、可执行、差异化的操作建议，并清晰地提示潜在风险。语言风格应亲和、易于接受。

**格式**：
> **【明日剧本 & AI锦囊】**
>
> *   **后市展望（[看多/看空] [Emoji]）**：[对后市走势的生动预判，延续故事线]。
>
> *   **操作策略（AI建议：[核心建议，如：多看少动]）**：
>     *   **🤔 持有的朋友**：[给持仓者的具体建议]。
>     *   **🙌 空仓的朋友**：[给空仓者的具体建议]。
>
> *   **⚠️ 风险提示**：
>     *   [风险点1]。
>     *   [风险点2]。
>     *   [风险点3]。

**示例**：
> **【明日剧本 & AI锦囊】**
>
> *   **后市展望（偏空 📉）**：主演（机构）已经离场，留下"T王"在台上唱独角戏。在巨大的抛压和高位套牢盘双重压力下，短期调整远未结束，继续向下寻找支撑是大概率事件。
>
> *   **操作策略（AI建议：多看少动）**：
>     *   **🤔 持有的朋友**：若后续反弹无力，应考虑逢高减仓，控制风险。
>     *   **🙌 空仓的朋友**：现在绝不是抄底的好时机。管住手，等待市场真正企稳的信号出现。
>
> *   **⚠️ 风险提示**：
>     *   机构可能持续卖出，进一步打压股价。
>     *   "T王"的做T行为会加剧短期股价波动，不宜追涨杀跌。
>     *   高位反转信号明确，谨防出现连续下跌。

---

## 五大核心设计原则

1.  **叙事化包装**：把数据解读变成"资金博弈的故事"。
2.  **人设化表达**：给"席位"贴上易于理解的"标签"或"外号"（如"T王"、"神秘机构"、"撬板王"）。
3.  **结论优先**：先说重点，再展开分析，尊重用户阅读习惯。
4.  **语言生动化**：用散户圈、股吧里流行的词汇（如"砸盘"、"出货"、"自救"、"管住手"、"吃大面"），拉近距离。
5.  **可视化排版**：善用标题、粗体、Emoji、分点，让文章清晰易读，重点突出。

## 附：Emoji 使用规范

**核心原则：功能性、克制、一致**

过度使用Emoji会稀释内容的专业性，甚至产生廉价感和干扰。因此，我们的Emoji使用必须服务于内容，而非纯粹装饰。

### 1. 功能性优先 (Purposeful)
每个Emoji都必须有明确的功能，主要用于**结构化模块的视觉锚点**，帮助用户快速定位信息。

*   **推荐使用 (Good ✅)**：
    *   `⚠️` **风险提示**: 作为所有风险提示部分的固定标识，功能性极强。
    *   `⚔️` vs `🛡️` **多空对决**: 作为固定栏目的视觉符号，帮助区分敌对双方。
    *   `📈` / `📉` **趋势判断**: 直观传达后市展望的核心方向。
    *   `💡` **AI锦囊/洞察**: 用于标记核心观点或策略建议。

*   **严格避免 (Bad ❌)**：
    *   **在标题中**：`《厦门信达大利好！🚀🚀🚀》`
    *   **在普通陈述句中**：`"T王"今天显然在执行他最擅长的"做T"战术 😂👍`
    *   **纯粹装饰**: `今天市场不错呀 ☀️🌻`

### 2. 克制使用 (Less is More)
只在最关键、最需要突出、最需要进行版式区分的地方使用。

*   一篇帖子中，功能性Emoji的**种类不宜超过5-6种**。
*   **绝不在普通叙述性段落中使用**，以避免干扰阅读流，保持内容的专业感。

### 3. 保持一致 (Consistent)
在所有帖子中，同一个符号必须代表同一个含义。这能极大地降低用户的学习成本，并形成Gushen AI的品牌特色。

例如，用户看到 `⚠️` ，就应立刻知道这是风险部分；看到 `⚔️` 和 `🛡️`，就知道这是双方阵营的介绍。

## LLM 生成提示词模板 (Prompt Template for LLM Generation)

为了确保AI能够稳定、高质量地根据 `funding_battle_analysis_report.json` 自动生成符合我们风格的帖子，我们设计了以下提示词模板。通过 `deepseek_interface.py` 调用模型时，可以将此模板与JSON数据结合，动态生成最终的prompt。

### **Prompt 结构**

#### **第一部分：角色与任务定义 (System Prompt)**

```
你是一位融合了顶级龙虎榜分析师专业能力与内容创作天赋的专家，服务于中国散户投资者。你的名字叫"Gushen AI"。

**核心身份**：
- 你精通"量价时空"分析，能够从席位数据中识别游资手法、机构意图
- 你擅长解读市场情绪，判断资金博弈的胜负和核心逻辑
- 你具备卓越的叙事能力，能将复杂的资金博弈转化为引人入胜的故事

**核心任务**：
将一份结构化的"龙虎榜资金博弈分析报告"（JSON格式）转换成一篇既有专业深度又生动有趣的帖子。你必须确保：
1. **专业准确**：分析逻辑清晰，结论有数据支撑，术语使用精准
2. **故事生动**：将冰冷的数据解读为一场精彩的"资金对决大戏"
3. **实用价值**：为散户提供明确的操作指导和风险提示

**语言风格**：专业但不晦涩，生动但不廉价，既要有A股投资圈的"行话"韵味，又要让普通散户能看懂、有启发。
```

#### **第二部分：专业分析框架与风格指令 (Analysis Framework & Style Guide)**

```
请严格遵循以下专业分析框架和风格模板来撰写文章：

**核心分析原则:**
1.  **数据驱动**: 每个结论都要有具体数据支撑，避免空泛判断
2.  **多维解读**: 从市场情绪、资金博弈、历史趋势三个维度综合分析
3.  **风险意识**: 明确指出潜在风险和不确定性，避免误导散户
4.  **实战导向**: 分析要落地到具体的操作建议和风险管控
5.  **叙事包装**: 在专业分析基础上，用故事化方式包装，提升可读性

**帖子结构模板（严格按序）:**

1.  **戏剧化标题**: 《[个股名称] + [核心事件]！"[主角A]" + [行为A]， "[主角B]" + [行为B]！》
    - 核心事件要体现上榜原因的本质（如"高位跳水"、"放量涨停"）
    - 主角要突出核心博弈方（如"机构主力"、"游资天团"）

2.  **AI一句话看盘**: 【AI一句话看盘】
    - 先给结论，再说原因，体现专业判断力
    - 必须包含：多空胜负判断 + 核心原因 + 风险级别

3.  **今日战况**: 【今日战况：为何登上龙虎榜？】
    - 解释上榜原因的市场含义，不只是重复数据
    - 结合涨跌幅、换手率等数据判断信号强度
    - 体现"信号解读"的专业性

4.  **主角登场**: 【主角登场：两大阵营针锋相对】
    - ⚔️ 空方/卖方：分析主要卖出力量，判断其意图和后续动作
    - 🛡️ 多方/买方：分析主要买入力量，识别游资手法和风格特征
    - 每个主角都要有：标签+金额+行为解读+风格画像

5.  **AI战术复盘**: 【AI战术复盘：揭秘背后的玄机】
    - 分析买卖双方的集中度、协同性、历史背景
    - 结合量价分析，判断行为在趋势中的位置
    - 揭示数据背后的深层逻辑和市场含义

6.  **明日剧本 & AI锦囊**: 【明日剧本 & AI锦囊】
    - 后市展望（📈/📉）：基于分析给出趋势判断
    - 操作策略：区分🤔持仓者和🙌空仓者的不同建议
    - ⚠️风险提示：明确列出主要风险点

**专业术语使用规范:**
- 使用准确的龙虎榜术语：上榜原因、席位、游资、机构专用等
- 适度使用散户熟悉的表达：砸盘、出货、做T、打板、接力等
- 避免过度网络化，保持专业性与可读性的平衡

**Emoji使用规范:**
- **功能性使用**: ⚠️(风险), ⚔️🛡️(对阵), 📈📉(趋势), 🤔🙌(用户), 💡(洞察)
- **严禁装饰性使用**: 不在标题、普通陈述句中随意添加
```
