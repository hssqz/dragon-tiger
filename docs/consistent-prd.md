### 4. 关注游资的连贯性：是“一日游”还是“锁仓接力”？

对于游资主导的妖股，这一点至关重要。

*   **一日游 (One-Day Trip):** 昨天上榜大笔买入的游资，今天就出现在了卖出席位上。
    *   **散户关注点：** **这是典型的短线砸盘信号**。如果散户昨天追高买入，今天就会面临巨大的抛压。这是追涨操作的大忌。

*   **锁仓 (Locking Position):** 昨天大买的游资，今天没有出现在卖出席位上。
    *   **散户关注点：** **这是一个积极信号**，表明该游资看好后市，选择格局和锁仓，股价短期内可能还有冲高的动能。

*   **接力 (Baton Pass):** 昨天的游资卖出，但榜上又出现了新的、实力同样强大的游资接盘买入。
    *   **散户关注点：** **这是行情可能延续的信号**。表明市场对该股的炒作热情不减，有新的资金愿意在高位承接，行情可能演变成“龙头妖股”。
