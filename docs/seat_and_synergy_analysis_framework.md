# 框架设计：从席位识别到协同小组分析

## 1. 核心理念

本框架旨在建立一套从**单个席位特征识别**到**协同小组深度剖析**的完整分析体系。我们的目标是穿透龙虎榜数据的表层，挖掘出背后资金的真实意图和关联关系，将"数据"转化为高价值的"情报"。

分析的核心思想，是两个层次的递进：
- **L1 - 微观层面**：精确刻画每一个上榜席位的**角色**、**风格**和**实力**。
- **L2 - 中观层面**：通过"三步漏斗模型"识别出真正的**"协同狼群"**，并对其**力量**、**意图**和**战术角色**进行定性、定量分析。

---

## 2. L1 - 单个席位识别与画像 (Seat Profiling)

在分析任何组合之前，我们必须先理解构成它的基本单元。

### 2.1 席位分类

首先，根据`player_info`中的信息，将席位分为四大基础类型：
- **知名游资**: 拥有明确的市场名号或派系（如"成都系"、"作手新一"）。这是分析的重点。
- **机构席位**: 名称中包含"专用"字样，代表公募、私募、社保、QFII等大资金。
- **量化席位**: 具备高频、小额、分散特征的席位（如"华鑫证券上海分公司"）。
- **普通席位**: 其他所有无法明确归类的席位。

### 2.2 关键指标解读

对于每个席位，必须关注以下核心指标：

- **净额 (`net_amount`)**: **方向信号之母**。净买入是做多，净卖出是做空。金额越大，意图越坚决。
- **净额占比 (`net_rate_pct`)**: **影响力指标**。净额占当天总成交额的比例，反映了该席位对股价的实际影响力。大于5%即为"核心影响者"。
- **交易活跃度**: 对比`buy_amount`和`sell_amount`。如果两者都很大但净额很小，则大概率为**做T席位**，其趋势指引意义有限，但表明个股流动性好。

### 2.3 风格画像 (Style Profiling)

通过`player_info.description`和`player_info.style`字段，结合历史行为，为每个席位（尤其是知名游资）打上风格标签：
- **操作手法**: 打板、低吸、追涨、做T、锁仓、砸盘。
- **偏好周期**: 超短线、短线、波段。
- **资金体量**: 亿元级、千万级。

## 3. L2 - 协同小组分析 (Synergy Group Analysis)

这是我们分析框架的灵魂。我们将通过一个**"三步漏斗模型"**来识别和分析协同小组。

![Synergy Analysis Funnel](https://i.imgur.com/3rQzW7a.png)

### **第一步：聚合 (Aggregation) - 找到所有潜在的"老乡"**

此阶段的目标是初步识别所有可能存在关联的席位。

1.  **同名聚合 (Highest Confidence)**
    - **逻辑**: 遍历所有上榜席位，如果它们的`player_info.name`字段完全相同（且非"普通席位"），则直接归为一个小组。这是最可靠的信号。
    - **例子**: 两个不同席位，`name`都是"T王"，则归为"T王"小组。

2.  **关联聚合 (Medium Confidence)**
    - **逻辑**: 依赖一个预先维护的"游资名人录"或"关联方图谱"。如果不同席位在图谱中被标记为关联方，则归为一组。
    - **例子**: "中信上海分公司"和"国泰君安上海江苏路"在图谱中都指向"徐翔系"，则归为一组。

3.  **行为聚合 (Lower Confidence, for Quants)**
    - **逻辑**: 识别交易行为高度相似的席位，通常用于发现量化基金。
    - **特征**: 同时在多只不相关股票上榜、单笔净额小、买卖频繁。

**实现方式**：在代码的"事实层"，我们优先实现**同名聚合**，因为它最准确、最易于实现。

### **第二步：计算 (Calculation) - 量化小组的"行动纲领"**

对聚合后的小组进行数据计算，这是后续分析的基础。

- **核心计算指标**:
  - `total_buy_amount_wan`: 小组总买入。
  - `total_sell_amount_wan`: 小组总卖出。
  - `net_amount_wan`: 小组净额。
  - `side`: 小组整体立场 (`long`/`short`/`neutral`)。
  - `seats_involved`: 席位列表。

### **第三步：甄别 (Filtering) - 过滤出"值得关注的狼群"**

不是所有小组都有分析价值。我们必须用严格的门槛过滤掉噪音。

1.  **规模门槛**: 小组成员数必须 **`>= 2`**。
2.  **资金门槛**: 小组总交易额（买+卖）应 **`>= 预设值`** (如2000万元)。
3.  **影响力门槛 (核心)**: 小组总交易额占当日总成交额比例应 **`>= 预设值`** (如3%)。

只有通过全部三层过滤的小组，才会被认定为真正的"协同小组"，并进入最终的LLM洞察分析环节。

---

## 4. 最终分析与输出

经过上述流程，我们将得到两层关键信息：
1.  **精确的席位画像**: 每个席位的类型、实力、风格。
2.  **高质量的协同小组**: 经过严格筛选的、真正有影响力的资金团体。

这两层信息共同构成了`StructuredFacts`，最终被送入LLM进行**意图定性**和**角色定位**的深度分析，产出`FundingBattleSummary`中高价值的`analysis`模块。这个框架确保了我们的分析既有广度（覆盖所有席位），又有深度（聚焦核心小组），并且每一步都有据可查。
